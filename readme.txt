Shipment Tracking Plugin for WooCommerce. Order tracking for various shipping services.

== Description ==

= Introduction =
WooCommerce Shipment Tracking Plugin helps to integrate tracking functionalities to your WooCommerce based store. By using this shipment tracking extension, you can provide your customers an easy way to track their shipments for various shipping services.

Our plugin gives the flexibility to easily add or remove shipping services providers from the admin settings. A complete list of pre-populated shipping services is available in the premium version mentioned below.

= How tracking feature works =
Before changing status of the order to ‘completed’, shop admin can add shipment tracking details by selecting Shipping Company and adding tracking id in the order details page. Once these details are added to an order, the tracking information will appear on the order view page in customers ‘My account’ section.

Apart from this, shop owner also can track the order status from the admin page.

Basic version features:
<ul>
<li>Option to set single shipping service for tracking, prefilled with USPS.</li>
<li>Set Tracking ID from order admin page ( excluding shipment date. )</li>
<li>Shipment Tracking status on customer's individual order page</li>
</ul>

== Installation ==

1. Upload the plugin folder to the ‘/wp-content/plugins/’ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. That's it – you can now configure the plugin.
